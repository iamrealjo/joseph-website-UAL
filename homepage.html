"HTML"

Hypertext Markup Language, better known as “HTML” is a structural coding framework that acts as the backbone for any web application, like this one. It's the framework responsible for creating things like our classes and elements that exist inside of the body. Think your buttons, the images, your text, they are all brought to life inside of html code.

" CSS"

“CSS”; cascading style sheets is the form of our web application. When we’re looking to design our elements with things like colour, opacity, or even more complexities like changes from hover to clicked state, this is the language we use. “CSS” style code can be written in alongside “HTML” to make specific changes to an element, but it’s good practice to create a separate “CSS” file that can be linked to numerous “HTML” pages. This allows us to maintain a consistent design language across all pages on our website.

Below are some examples of the “CSS" code used on this site. Try them out in your next projects
